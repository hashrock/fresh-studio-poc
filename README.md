# fresh project

### Usage

Start the project:

```
deno task start
```

# これは何

- freshでscaffolder兼ドキュメント

# 機能

- routeの作成（＋カスタムハンドラの追加など）
- apiの作成（ありがちなDBアクセスなどのコード例も入れてくれるとなおよい）
- コンポーネントストアからの検索・追加
- 404ページから該当するrouteの生成
- 要らなくなったときに簡単にEjectする機能
